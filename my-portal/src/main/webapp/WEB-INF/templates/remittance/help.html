Login Section

How do I install Indepay portal?

Great News! You don’t need to install Indepay. Just open remit.indepay.com in your web browser and enter your login details for Indepay and start transacting from anywhere.

How can I login to Indepay portal?

Simply visit remit.indepay.com Enter your login details for Indepay and click on the “Login” button. If you are still not able to login, then please get in touch with Indepay Customer Care.

The Indepay system is not allowing me to log on. What should I do?

Make sure that you are accessing Indepay on remit.indepay.com Check your login details. If you are still not able to login, then please get in touch with Indepay Customer Care.

How can I request a new password?

Please click on “Forget Password”. Enter your Indepay registered Mobile number & validate OTP, post which you can set your new password. In case you face any problem please get in touch with Indepay customer care.

I am unable to use Indepay after I login?

Please check if you are using the latest version of web browser or else logout and login again. Check if you also have sufficient balance/limit in your wallet. If the issue persists, please get in touch with the Indepay Customer Care
